## 唯一突破

**v5 日线 LSTM：Test IC3=0.114。** 866K 日线预测，横截面有效。

月度回归预测（1m/3m/6m 前瞻收益）全部失败——Test IC 从未超过 0.03（除 v1 的 0.025）。

## 根因分析

1. **Regime change**：2015-2021 训练的模式不迁移到 2024-2026。A 股市场结构在过去 10 年发生了根本性变化（注册制、外资准入、量化基金占比上升）
2. **月线信号退化**：60 个月 lookback 中，最近 6-12 个月的信息占主导，早期历史信息几乎无贡献
3. **价量 alpha 已被吸收**：21 维技术指标（MA/MACD/RSI/KDJ 等）是公开信息，已被市场定价
4. **小盘股反向信号**：CSI500 Test IC=-0.08——小盘股的价量信号与大盘股方向相反，等权聚合时信号抵消

## 结论

**月线频率 + 大盘 universe + 价量特征 = LSTM 无法产生 out-of-sample 预测力。**

这不是架构问题（试了 22 种架构），不是特征问题（试了 21/33/40 维），不是频率问题（试了月/周/日），不是 universe 问题（试了 HS300/CSI500）。是**数据本身的信息上限**。
