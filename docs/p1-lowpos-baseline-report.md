## 2. 反转因子基线

```
bullish (跌最深20%):  WinsMean =   6.14%
bearish (跌最浅20%):  WinsMean =  38.60%
spread:                        -32.46%
95% CI:              [-65.54%, -15.08%]
nBlocks:             573
```

**反转因子是反向的。** 在低位池中"越跌越买"的股票表现远逊于"跌得不深"的。这与价值 trap 假说一致——最深的折扣往往对应基本面问题，不是买入机会。
